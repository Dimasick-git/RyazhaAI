═══════════════════════════════════════════════════════════════
ШАГ 1: СОЗДАЙ АККАУНТ НА GITHUB
═══════════════════════════════════════════════════════════════

1. Открой: https://github.com/signup

2. Введи email → Continue

3. Создай пароль → Continue

4. Придумай username → Continue

5. Подтверди email (придет письмо)

✅ Готово! У тебя есть GitHub аккаунт!


═══════════════════════════════════════════════════════════════
ШАГ 2: СОЗДАЙ РЕПОЗИТОРИЙ
═══════════════════════════════════════════════════════════════

1. Зайди на: https://github.com/new

2. Заполни:
   Repository name: ryazha-ai
   Description: RYAZHA AI - Умный помощник для Nintendo Switch CFW
   Public ✅ (чтобы все видели!)
   
3. Нажми большую зеленую кнопку:
   "Create repository"

✅ Репозиторий создан!


═══════════════════════════════════════════════════════════════
ШАГ 3: ЗАГРУЗИ ФАЙЛЫ ПРОЕКТА
═══════════════════════════════════════════════════════════════

На странице репозитория увидишь текст:
"Quick setup — if you've done this kind of thing before"

Под ним есть ссылка:
"uploading an existing file"

1. Нажми "uploading an existing file"

2. Откроется страница загрузки

3. Открой папку проекта:
   c:\Users\dimas\OneDrive\Рабочий стол\AI RYAHA

4. Выдели ВСЕ файлы (Ctrl+A)
   ⚠️ НЕ ЗАГРУЖАЙ папку node_modules (если есть)!
   
5. Перетащи файлы в окно браузера
   (где написано "Drag files here")

6. Подожди загрузки (может занять 1-2 минуты)

7. Внизу страницы в поле "Commit changes" напиши:
   "Initial commit - RYAZHA AI v2.0"

8. Нажми зеленую кнопку:
   "Commit changes"

✅ Файлы загружены на GitHub!


═══════════════════════════════════════════════════════════════
ШАГ 4: ПОДКЛЮЧИ NETLIFY
═══════════════════════════════════════════════════════════════

1. Открой: https://app.netlify.com/signup

2. Нажми большую кнопку:
   "GitHub" (войти через GitHub)

3. Появится окно от GitHub:
   "Authorize netlify"
   
4. Нажми зеленую кнопку:
   "Authorize netlify"

✅ Вошел в Netlify через GitHub!


═══════════════════════════════════════════════════════════════
ШАГ 5: ЗАДЕПЛОЙ САЙТ
═══════════════════════════════════════════════════════════════

1. На главной странице Netlify нажми:
   "Add new site" → "Import an existing project"

2. Выбери:
   "Deploy with GitHub"

3. Появится список твоих репозиториев
   Найди: ryazha-ai
   Нажми на него

4. На странице "Site settings" заполни:

   Branch to deploy: main (или master)
   
   Build command: npm run build
   
   Publish directory: dist
   
   ⚠️ ВАЖНО! Эти настройки должны быть точно такими!

5. Нажми большую кнопку:
   "Deploy ryazha-ai"

6. ПОДОЖДИ 2-3 МИНУТЫ!
   
   Увидишь страницу с логами сборки.
   
   Будет написано:
   "Site deploy in progress"
   
   Когда закончится, появится:
   "Site is live" ✅

✅ САЙТ ЗАДЕПЛОЕН!


═══════════════════════════════════════════════════════════════
ШАГ 6: ПОЛУЧИ ССЫЛКУ!
═══════════════════════════════════════════════════════════════

После успешного деплоя увидишь ссылку вверху страницы:

   https://случайное-имя-123456.netlify.app

ЭТО ТВОЯ ПУБЛИЧНАЯ ССЫЛКА! 🎉

Можешь переименовать:

1. Нажми "Site settings"

2. Найди "Site information"

3. Нажми "Change site name"

4. Введи: ryazha-ai

5. Save

Теперь ссылка будет:
   https://ryazha-ai.netlify.app

═══════════════════════════════════════════════════════════════
❌ ПРОБЛЕМЫ И РЕШЕНИЯ
═══════════════════════════════════════════════════════════════

Проблема: Деплой failed (не удался)

Решение:
   • Проверь что Build command: npm run build
   • Проверь что Publish directory: dist
   • Проверь что загрузил package.json
   • Попробуй заново

Проблема: Сайт показывает 404

Решение:
   • Проверь настройку Publish directory: dist
   • Попробуй задеплоить заново

Проблема: AI не отвечает

Решение:
   • Это нормально! Cody API может иметь задержку
   • Подожди 10-15 секунд
   • Или используются fallback ответы
