🔑 API КЛЮЧИ (ОПЦИОНАЛЬНО):
═══════════════════════════════════════════════════════════════

БЕЗ API:
• Сайт работает в демо-режиме ✅
• AI дает умные ответы на Switch темы ✅
• Всё работает из коробки! ✅

С HUGGING FACE API (для полного AI):
• Более умные и развернутые ответы
• Понимает сложные вопросы
• БЕСПЛАТНО! Регистрация 30 секунд

КАК ПОЛУЧИТЬ:
1. https://huggingface.co → Sign up
2. Settings → Access Tokens → New token
3. Скопируй токен
4. Открой .env.local в папке проекта
5. Вставь токен после VITE_HF_TOKEN=
6. Сохрани файл
7. Перезапусти ПРОСТО_ЗАПУСТИ_МЕНЯ.bat

🌐 ДЕПЛОЙ В ИНТЕРНЕТ:
═══════════════════════════════════════════════════════════════

ВАРИАНТ 1 - ПРОСТОЙ (Drag & Drop):
1. Дважды кликни: ДЕПЛОЙ_NETLIFY.bat
2. Подожди сборки
3. Зайди на https://app.netlify.com/drop
4. Перетащи папку dist/ на страницу
5. Получи URL! 🎉

ВАРИАНТ 2 - АВТОМАТИЧЕСКИЙ:
1. npm install -g netlify-cli
2. netlify login
3. netlify deploy --prod --dir=dist

💡 ЧАСТЫЕ ВОПРОСЫ:
═══════════════════════════════════════════════════════════════

❓ Что делать после установки Node.js?
✅ Закрой ВСЕ окна CMD/PowerShell, запусти .bat снова

❓ Ошибка при npm install?
✅ Открой PowerShell в папке проекта:
   npm cache clean --force
   npm install

❓ Порт 3000 занят?
✅ Vite автоматически выберет другой (3001, 3002...)

❓ Нужны ли API ключи?
✅ НЕТ! Демо-режим работает отлично!
   API ключи - для более умного AI (опционально)

❓ Как остановить сервер?
✅ Нажми Ctrl+C или закрой окно .bat
