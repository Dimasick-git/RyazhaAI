# 🚀 Быстрая настройка Docker Hub (2 минуты)

## 1️⃣ Регистрация Docker Hub
- 🔗 https://hub.docker.com/signup
- 📧 Введи email и создай аккаунт
- 👤 Запомни свой username (например: `dimasick`)

## 2️⃣ Добавь Secrets в GitHub
- 🔗 https://github.com/Dimasick-git/RyazhaAI/settings/secrets/actions
- ➕ "New repository secret"

**Secret 1:**
- Name: `DOCKER_USERNAME`
- Value: твой Docker Hub username

**Secret 2:** 
- Name: `DOCKER_PASSWORD`
- Value: твой Docker Hub пароль

## 3️⃣ Готово! ✅
Теперь GitHub Actions будет автоматически собирать Docker образы!
