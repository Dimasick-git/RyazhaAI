⚡ БЕСПЛАТНЫЙ БЕЗЛИМИТНЫЙ AI - КАК ЭТО РАБОТАЕТ:
═══════════════════════════════════════════════════════════════

🤖 Cody API:
   • Endpoint: https://cody.su/api/v1
   • Модель: GPT-4.1 (бесплатно!)
   • Без лимитов на запросы!
   • Zero-retention (приватность!)

🔄 Автоматизация:
   1. Ты задаешь вопрос в чате →
   2. RYAZHA AI автоматически получает ключ от Cody API →
   3. Отправляет запрос к GPT-4.1 →
   4. Получаешь умный ответ! ✅
   
   📦 Ключ кэшируется - не нужно получать каждый раз!
   🔄 Если устареет - автоматически обновится!

💡 Fallback:
   Если Cody API недоступен → 
   используются умные заготовленные ответы (15+)
