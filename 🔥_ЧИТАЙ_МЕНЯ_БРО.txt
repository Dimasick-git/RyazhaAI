Деплой через GitHub + Netlify = Публичная ссылка для всех!


📋 3 ПРОСТЫХ ШАГА:
═══════════════════════════════════════════════════════════════


ШАГ 1: ЗАГРУЗИ НА GITHUB
────────────────────────────────────────────────────────────

1. Зайди: https://github.com/new
   (Если нет аккаунта - сделай, это бесплатно!)

2. Заполни:
   • Repository name: ryazha-ai
   • Public ✅
   • Нажми "Create repository"

3. На странице репозитория нажми:
   "uploading an existing file"

4. Открой папку:
   c:\Users\dimas\OneDrive\Рабочий стол\AI RYAHA

5. Выдели ВСЕ файлы (Ctrl+A)
   ⚠️ НЕ загружай node_modules если есть!

6. Перетащи в браузер

7. Внизу напиши: "Initial commit"

8. Нажми "Commit changes"

✅ Файлы на GitHub!


ШАГ 2: ПОДКЛЮЧИ NETLIFY
────────────────────────────────────────────────────────────

1. Зайди: https://app.netlify.com/signup

2. Нажми "GitHub" (войти через GitHub)

3. Authorize netlify

4. Нажми "Add new site" → "Import an existing project"

5. "Deploy with GitHub"

6. Выбери репозиторий: ryazha-ai

7. Настройки:
   Build command: npm run build
   Publish directory: dist

8. "Deploy ryazha-ai"

9. ЖДИ 2-3 МИНУТЫ!

✅ Деплой идет!


ШАГ 3: ПОЛУЧИ ССЫЛКУ!
────────────────────────────────────────────────────────────

Когда увидишь "Site is live":

   https://что-то.netlify.app

ЭТО ТВОЯ ПУБЛИЧНАЯ ССЫЛКА! 🎉

Можешь переименовать:
   Site settings → Change site name → ryazha-ai

Станет:
   https://ryazha-ai.netlify.app

❓ ПРОБЛЕМЫ?
═══════════════════════════════════════════════════════════════

   → Проверь Build command: npm run build
   → Проверь Publish directory: dist

Сайт не загружается?
   → Подожди еще 1-2 минуты
   → Обнови страницу
