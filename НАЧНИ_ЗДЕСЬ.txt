2. Открой PowerShell и выполни:

🔑 НАСТРОЙКА (ОПЦИОНАЛЬНО):

Для полноценного AI чата получи бесплатный токен:

1. Зайди на huggingface.co
2. Зарегистрируйся (1 минута)
3. Settings → Access Tokens → New token
4. Скопируй токен
5. Добавь в src/services/api.js

Подробно: API_SETUP.md

Без токена работает демо-режим с заготовленными ответами!

─────────────────────────────────────────────────────────────

🌐 ДЕПЛОЙ НА ХОСТИНГ:

После тестирования задеплой на Netlify (бесплатно):

1. npm run build
2. Зайди на netlify.com
3. Drag & drop папку dist/
4. Получи URL типа: your-site.netlify.app

Подробно: DEPLOYMENT.md

─────────────────────────────────────────────────────────────

🎮 СБОРКА ДЛЯ SWITCH:

Если есть DevkitPro:

1. Задеплой сайт (см. выше)
2. Обнови URL в switch-app/source/main.cpp
3. cd switch-app
4. make
5. Скопируй ai-ryaha.nro на SD карту Switch
6. Запусти через Homebrew Menu

Подробно: switch-app/README.md
