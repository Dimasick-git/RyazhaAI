Установка на Nintendo Switch

Требования:

  • Nintendo Switch с кастомной прошивкой (Atmosphere/Hekate)
  • SD карта
  • Подключение к интернету

Шаг 1: Скачайте файл

Скачайте файл ai-chat.nro с GitHub.

Шаг 2: Установка

1. Скопируйте файл ai-chat.nro на SD карту Switch

2. Путь на SD карте: /switch/ai-chat/ai-chat.nro

3. Вставьте SD карту в Switch

Шаг 3: Запуск

1. Запустите Homebrew Menu (удерживайте R при запуске игры/альбома)

2. Найдите "AI Chat" в списке

3. Запустите приложение

4. Switch откроет браузер с AI чатом!

Ссылка на скачивание прошивки: GitHub Releases

Решение проблем

Приложение не запускается на Switch

Решение:

  • Убедитесь, что Switch имеет кастомную прошивку (Atmosphere)
  • Проверьте путь к файлу: /switch/ai-chat/ai-chat.nro
  • Попробуйте перезапустить Switch

Браузер не открывается

Решение:

  • Проверьте подключение к интернету
  • Убедитесь, что DNS настроен правильно
  • Попробуйте открыть браузер вручную через настройки

AI не отвечает

Решение:

  • Проверьте подключение к интернету
  • Попробуйте обновить страницу
  • Если проблема сохраняется - попробуйте другой API ключ
