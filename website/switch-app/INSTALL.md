# Установка AI Chat на Nintendo Switch

## ⚠️ Важно: Нужен Switch с кастомной прошивкой (CFW)

## Вариант 1: Использовать готовый NX-Shell (РЕКОМЕНДУЕТСЯ)

### Шаг 1: Установите NX-Shell
1. Скачайте NX-Shell: https://github.com/joel16/NX-Shell/releases
2. Скопируйте `NX-Shell.nro` на SD карту в `/switch/NX-Shell/`
3. Вставьте SD карту в Switch

### Шаг 2: Создайте закладку
1. Запустите NX-Shell на Switch
2. Нажмите кнопку Settings
3. Добавьте закладку с вашим URL
4. Готово! Теперь можно открывать AI Chat одной кнопкой

## Вариант 2: Использовать Linkalho

### Шаг 1: Установите Linkalho
1. Скачайте: https://github.com/rdmrocha/linkalho/releases
2. Запустите на ПК
3. Подключите Switch через USB

### Шаг 2: Создайте форвардер
1. Откройте Linkalho
2. Выберите "Create Web Forwarder"
3. Введите URL: `https://ваш-сайт.netlify.app/simple.html`
4. Название: "AI Chat"
5. Нажмите Create
6. Установится автоматически на Switch

## Вариант 3: Готовый .nro файл

К сожалению, я не могу создать готовый .nro без компилятора.

### Где взять готовый .nro:

**Опция A: Попросите кого-то скомпилировать**
- Отправьте папку `C:\website\switch-homebrew` другу с DevkitPro
- Или попросите на форумах: gbatemp.net, reddit.com/r/SwitchHacks

**Опция B: Скомпилируйте сами (15 минут)**
1. Установите DevkitPro: https://github.com/devkitPro/installer/releases
2. Откройте MSYS2 (из меню DevkitPro)
3. Выполните:
```bash
cd /c/website/switch-homebrew
make
```
4. Получите `ai-chat.nro`

**Опция C: Используйте онлайн компилятор**
- GitHub Actions может скомпилировать автоматически
- Создайте репозиторий и загрузите код

## Вариант 4: Самый простой - DNS трюк (БЕЗ .nro)

### Без установки приложений:

1. **На Switch:**
   - Настройки → Интернет
   - Выберите WiFi → Изменить настройки
   - DNS настройки → Вручную
   - Первичный DNS: `045.055.142.122`
   - Сохранить

2. **Попробуйте подключиться:**
   - Switch попытается проверить соединение
   - Откроется браузер
   - Введите ваш URL

3. **Или через игру:**
   - Запустите любую игру с интернет-функциями
   - Найдите место, где открывается браузер
   - Введите URL

## Размещение сайта (ОБЯЗАТЕЛЬНО)

Перед использованием на Switch, разместите сайт онлайн:

### Netlify (30 секунд):
1. Откройте: https://app.netlify.com/drop
2. Перетащите `C:\website\standalone\simple.html`
3. Переименуйте в `index.html` перед загрузкой
4. Получите URL типа: `https://random-name.netlify.app`

### GitHub Pages:
1. Создайте репозиторий на github.com
2. Загрузите `simple.html` как `index.html`
3. Settings → Pages → Enable
4. URL: `https://username.github.io/repo-name`

## Настройка API ключа

1. Откройте сайт на компьютере/телефоне
2. Перейдите в "⚙️ Настройки"
3. Получите ключ: https://api.chatanywhere.tech/v1/oauth/free/render
4. Вставьте и сохраните
5. Теперь можно использовать на Switch

## Проверка

1. Откройте URL на телефоне - должен работать
2. Только потом пробуйте на Switch
