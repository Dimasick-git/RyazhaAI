🚀 СЛЕДУЮЩИЕ ШАГИ:

1️⃣ УСТАНОВИ NODE.JS:
   📥 Скачай: https://nodejs.org (LTS версия)
   ⚙️ Установи с галкой "Add to PATH"
   ✅ Проверь: node --version

2️⃣ УСТАНОВИ ЗАВИСИМОСТИ:
   cd "c:\Users\dimas\OneDrive\Рабочий стол\AI RYAHA"
   npm install

3️⃣ ЗАПУСТИ ЛОКАЛЬНО:
   npm run dev
   Открой: http://localhost:3000

4️⃣ ЗАДЕПЛОЙ НА NETLIFY:
   
   Вариант A - Git:
   • Залей на GitHub
   • Подключи к Netlify
   • Автодеплой готов!
   
   Вариант B - Drag & Drop:
   • npm run build
   • Зайди на netlify.com
   • Перетащи папку dist/

5️⃣ ОБНОВИ URL В SWITCH ПРИЛОЖЕНИИ:
   • Открой: switch-app/source/main.cpp
   • Строка 61: websiteUrl = "твой URL"
   • Сохрани

6️⃣ СОБЕРИ .NRO (если есть DevkitPro):
   cd switch-app
   make
   
   Результат: ryazha-ai.nro
