    ╚═════════════════════════════════════════════════════════════════╝

    Для полноценного AI чата (РЕКОМЕНДУЕТСЯ):

    1. Зайди: https://huggingface.co
    2. Зарегистрируйся (бесплатно!)
    3. Settings → Access Tokens → New token
    4. Скопируй токен
    5. Открой: src/services/api.js
    6. Добавь токен в код (там инструкция)

    📖  Подробно в: API_SETUP.md

    ⚠️  Без API работает демо-режим с заготовленными ответами!

    После тестирования задеплой на Netlify (БЕСПЛАТНО):

    Способ 1 - Drag & Drop (самый простой):
        1. npm run build
        2. Зайди: https://netlify.com
        3. Drag & drop папку dist/
        4. Готово! 🎉

    Способ 2 - Через Git (рекомендуется):
        1. Залей проект на GitHub
        2. Подключи к Netlify
        3. Автодеплой при каждом push!

    📖  Подробно в: DEPLOYMENT.md

    ║               🎮  СБОРКА ДЛЯ NINTENDO SWITCH  🎮                ║
    ╚═════════════════════════════════════════════════════════════════╝

    Требования:
        • Nintendo Switch с CFW (Ryazhenka/Atmosphere)
        • DevkitPro (для сборки)
        • Задеплоенный сайт

    Инструкция:

        1. Задеплой сайт (см. выше)
        
        2. Обнови URL в switch-app/source/main.cpp
        
        3. Собери .nro:
           cd switch-app
           make
        
        4. Скопируй ai-ryaha.nro на SD → /switch/
        
        5. Запусти через Homebrew Menu!

    📖  Подробно в: switch-app/README.md

    Если что-то не работает:

    1. ❓ "npm не найден"
       → Установи Node.js с nodejs.org

    2. ❓ Ошибки при npm install
       → Попробуй: npm cache clean --force

    3. ❓ Порт 3000 занят
       → Vite автоматически выберет другой (3001, 3002...)

    4. ❓ AI не отвечает
       → Демо-режим работает без API
       → Для полного AI добавь Hugging Face токен

    5. ❓ Другие проблемы
       → Читай документацию в папке
       → Проверь консоль браузера (F12)
